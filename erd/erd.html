ColorAndShades ERD

ERD Description

Table PrimaryColor

  • This table shows primary color names and identifying numbers. It is used to sort primary colors.
  • The Id attribute gives each color a unique number for identification.
  • The PrimaryColorName attribute gives color names.
  • This table has one primary key Id and it is used a foreign key in the Shades table(PrimaryColorId).

Table Shades

  • This table is used to sort color shades and see what primary colors they connect with using the PrimaryColorId attribute.
  • The Id attribute gives each shade a unique number for identification.
  • PrimaryColorId attribute is used to connect primary color to a shade.
  • ColorName attribute is used to know what color shade there is.
  • This table uses primary key PrimaryColorId from PrimaryColor table as a foreign key to connect shade to a color. Shades can have one primary color and colors can have many shades
