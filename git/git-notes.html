    Commands for Git

    • $git init - Initializes a local Git repository/ creates a repository. Makes git files to folder. Before that you have to navigate to project folder.
    • $git status - Show status of git repository and shows if we have any untracked changes.
    • $git add . - Add all files/ changes of your project to tracking logic of git.
    • $git add filename - Add selected files with it, but typically we use git add. to add all files.
    • $git commit -m "comment" - Commited code and added to repository, "comment" is to understand what changes you made.
    • $git branch - Shows all the branches that are available and shows what branch you are currently using.
    • $git checkout -b new-branch-name - Creates new branch with name new-branch-name
    • $git log - Allows to get overview of all the commits that we have in the branch that we are currently located in. Last commit you made is HEAD in your branch. You can get commit id from here.
    • $git checkout commitId - You can use it to do both, check out branches or in that case, commits. And when doing git log, you will get the ID of a specific commmit and then get checkout plus ID to check out this commit. If you want to go back to the latest commit then you have to write git master
    • $git reset --hard commitId - Resets the index and working tree. Any changes to tracked files in the working tree since commit are discarded. Instead of commitId should be the ID we want to go back.
    • $git checkout -- . - Helps to go back to the last commit we did and to get rid of all unstaged changes. Means that you have saved the file but have not used add .
    • $git checkout branch-name - Changes to this branch.
    • $git merge branch-name - Add files from other branch to the master branch. If you want to merge the branches you have to be in the master branch and to add the changes from the other branch.
    • $git branch -D branch-name - Now when we have merged branches we dont need the other branch anymore and we can delete it

    Commands for GitHub

    • $git remote add origin ... - To establish a connection between the local repository and a remote repository on GitHub.
    • $git remote - List the remote connections you have to other repositories.
    • $git remote -v - List the current remotes associated with the local repository. To see the origin more specifically and the url behind that.
    • $git push -u origin master - Push your branches to the remote repository
    • $git branch - Shows us in which branch we are currently.
    • $git clone ... - To clone someone's repository to our computer.
    • $git push - Push the files. Uploads all local branch commits to the remote.
    • $git fetch - To synchronize the code in GitHub repository and to remote tracking branch.
    • $git merge origin/master - To merge the local repository with the remote tracking branch.
    • $git pull - Updates your current local working branch with all new commits from the corresponding remote branch on GitHub. Git pull is a combination of git fetch and git merge.
    • $git remote rm origin - Remove the repository.
